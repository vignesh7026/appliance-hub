# 🛒 Appliance Hub

Appliance Hub is a full-stack e-commerce web application developed for buying home appliances online. It provides customers with a simple and user-friendly shopping experience while allowing administrators to manage products and orders efficiently.

## 🚀 Features

- User Registration & Login
- Product Search
- Category-wise Product Listing
- Product Details Page
- Shopping Cart
- Checkout System
- Order Management
- Admin Dashboard
- Responsive Design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/Appliance-Hub.git
```

### Navigate to Project

```bash
cd Appliance-Hub
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Install Frontend Dependencies

```bash
cd ../client
npm install
```

### Configure Database

1. Install MySQL.
2. Create a database named:

```
appliancehub
```

3. Import the SQL file located inside the `database` folder.

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

The application will run on:

Frontend:
```
http://localhost:3000
```

Backend:
```
http://localhost:5000
```
